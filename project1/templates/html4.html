Django Comments Information

startproject <projectname>

Creates a new Django project with the given name.

runserver

Starts the Django development server.

startapp <appname>

Creates a new Django application with the given name.

migrate

Synchronizes the database state with the current set of models and migrations.

makemigrations

Creates new migrations based on the changes detected to your models.

showmigrations

Shows all migrations in your project.

createsuperuser

Creates a superuser account to access the Django admin interface.

shell

Opens the Django interactive Python shell.
